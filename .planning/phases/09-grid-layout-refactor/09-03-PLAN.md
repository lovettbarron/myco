Implement `collect_dividers`:
```rust
fn collect_dividers(
    node: &SplitNode,
    grid: &GridLayout,
    out: &mut Vec<Divider>,
) {
    if let SplitNode::Branch { direction, children, taffy_node, .. } = node {
        for i in 0..children.len().saturating_sub(1) {
            // Get the rect of child[i] (may be a branch -- need subtree bounding rect)
            let child_rect = get_subtree_rect(&children[i], grid);
            let (cx, cy, cw, ch) = child_rect;

            let (orientation, position, extent_start, extent_end) = match direction {
                SplitDirection::Horizontal => {
                    // Vertical divider at right edge of child[i]
                    (Orientation::Vertical, cx + cw, cy, cy + ch)
                }
                SplitDirection::Vertical => {
                    // Horizontal divider at bottom edge of child[i]
                    (Orientation::Horizontal, cy + ch, cx, cx + cw)
                }
            };

            out.push(Divider {
                orientation,
                position,
                extent_start,
                extent_end,
                container_node: *taffy_node,
                child_index: i,
                constrained: false,
            });
        }

        // Recurse into children (adds deeper dividers after parent ones)
        for child in children {
            collect_dividers(child, grid, out);
        }
    }
}
```

Implement `get_subtree_rect` -- for a leaf, returns `grid.get_panel_rect(taffy_node)`. For a branch, returns the bounding rect of all leaf descendants:
```rust
fn get_subtree_rect(node: &SplitNode, grid: &GridLayout) -> (f32, f32, f32, f32) {
    match node {
        SplitNode::Leaf { taffy_node, .. } => grid.get_panel_rect(*taffy_node),
        SplitNode::Branch { taffy_node, .. } => {
            // Use the taffy layout of the branch node itself
            grid.get_panel_rect(*taffy_node)
        }
    }
}
```
NOTE: `get_panel_rect` walks to root accumulating offsets. For a Branch node, it should work the same way since taffy computes layout for all nodes including containers. Verify this works in tests.

**Step 5: Rewrite hit_test_divider with extent checking:**
```rust
pub fn hit_test_divider(
    dividers: &DividerSet,
    cursor_x: f32,
    cursor_y: f32,
) -> Option<(usize, Orientation)> {
    let half_zone = DIVIDER_HIT_ZONE / 2.0;
    // Dividers are ordered deepest-first (from the reverse in compute_dividers)
    // First match wins = deepest container priority per D-09
    for (i, div) in dividers.dividers.iter().enumerate() {
        match div.orientation {
            Orientation::Vertical => {
                if (cursor_x - div.position).abs() <= half_zone
                    && cursor_y >= div.extent_start
                    && cursor_y <= div.extent_end
                {
                    return Some((i, Orientation::Vertical));
                }
            }
            Orientation::Horizontal => {
                if (cursor_y - div.position).abs() <= half_zone
                    && cursor_x >= div.extent_start
                    && cursor_x <= div.extent_end
                {
                    return Some((i, Orientation::Horizontal));
                }
            }
        }
    }
    None
}
```

Implementation:
1. Get the container's rect to determine the total size in the drag direction.
2. Compute the current pixel sizes of `child[child_index]` and `child[child_index + 1]` from their flex_grow weights.
3. Convert delta_pixels to weight delta.
4. Apply delta, clamping so neither child's resulting pixel size is below `PANEL_MIN_WIDTH` (for horizontal containers / vertical dividers) or `PANEL_MIN_HEIGHT` (for vertical containers / horizontal dividers).
5. If clamped, return `true` (constrained).
6. Normalize weights in the container.
7. Update taffy styles: `grid.tree_mut().set_style(child_node, leaf_style(new_weight))` for both affected children.
8. Return `false` (unconstrained).
