<objective>
Replace the CSS Grid 2-level layout model with a recursive N-ary split tree backed by taffy Flexbox.

Purpose: The current model only supports columns at root + vertical stacking within columns. This plan introduces the SplitNode tree data structure that enables arbitrary nesting, and rewires GridLayout to use Flexbox instead of CSS Grid. After this plan, the app works exactly as before but the internal representation supports N-ary recursive splits.

Output: `src/grid/tree.rs` (new), modified `src/grid/layout.rs` and `src/grid/mod.rs`. All existing grid tests pass (rewritten for new model).
</objective>

    let mut tree = TaffyTree::new();
    let panel = tree.new_leaf(Style {
        flex_grow: 1.0,
        flex_shrink: 1.0,
        flex_basis: Dimension::Length(0.0),

**Step 3: Fix get_panel_rect walk-to-root bug (RESEARCH.md Pitfall 1):**
Replace single `if let Some(parent)` with a `while let` loop that accumulates offsets up to root:
```rust
pub fn get_panel_rect(&self, node: NodeId) -> (f32, f32, f32, f32) {
    let layout = self.tree.layout(node).unwrap();
    let mut x = layout.location.x;
    let mut y = layout.location.y;
    let w = layout.size.width;
    let h = layout.size.height;

    let mut current = node;
    while let Some(parent) = self.tree.parent(current) {
        if parent == self.root {
            break;
        }
        let parent_layout = self.tree.layout(parent).unwrap();
        x += parent_layout.location.x;
        y += parent_layout.location.y;
        current = parent;
    }

    (x, y, w, h)
}
```
