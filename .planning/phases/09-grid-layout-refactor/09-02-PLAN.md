3. `fn split_in_tree(tree: &mut SplitNode, target_panel_id: PanelId, direction: SplitDirection, new_panel_id: PanelId, taffy: &mut TaffyTree<()>) -> bool` -- the core recursive split algorithm per RESEARCH.md Example 1. Key logic:
   - If current node is a Leaf matching target: replace it with a Branch containing the old leaf and new leaf (50/50 weights). Create new taffy nodes with leaf_style and container_style.
   - If current node is a Branch with same direction as split AND the target is a direct child leaf: flatten -- insert new leaf as sibling after target. Redistribute weights equally (1/n for n children). Update taffy children and styles.
   - Otherwise: recurse into the child containing the target.
   - Important: When creating a new Branch that replaces a Leaf, the new Branch's taffy_node must REPLACE the old leaf's taffy_node in the parent's children list. Detach old leaf from parent, create container with old+new as children, attach container to parent at same position.

4. `enum RemoveResult { NotFound, Removed, Collapse(SplitNode) }` and `fn remove_from_tree(tree: &mut SplitNode, target: PanelId, taffy: &mut TaffyTree<()>) -> RemoveResult` -- per RESEARCH.md Example 2. Key logic:
   - Leaf matching target: return Collapse (caller replaces this node).
   - Branch: recurse into children. If child returns Collapse, remove that child and its weight. If 1 child remains, return Collapse(survivor) for parent to handle (D-03 auto-unwrap). Otherwise normalize weights and rebuild taffy children.
   - Handle taffy cleanup: `taffy.remove(node)` for removed nodes, `taffy.set_children(branch_taffy, &child_nodes)` after removals.

fn migrate_v1_to_v2(layout: &LayoutConfig) -> TreeLayoutConfig {
    let children: Vec<TreeNodeConfig> = layout.columns.iter().map(|col| {
        match col {
            ColumnConfig::Single(cap) => TreeNodeConfig::Leaf {
                cap: cap.clone(),
                weight: 1.0,

            ColumnConfig::Stack { caps } => {
                let leaves: Vec<TreeNodeConfig> = caps.iter().map(|cap| {
                    TreeNodeConfig::Leaf { cap: cap.clone(), weight: 1.0 }
                }).collect();
                let n = leaves.len();
                TreeNodeConfig::Branch {
