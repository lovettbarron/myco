<objective>
Implement project configuration persistence: serialize the grid layout to `.myco/config.json` with debounced auto-save, and restore it on application launch. Also create `~/.myco/preferences.json` for global defaults.

Purpose: After this plan, closing and reopening Myco in a project folder restores the exact panel arrangement (terminal CWD, canvas files, markdown files) from last session. This is the core value of Phase 5.

Output: `src/config/` module with ProjectConfig and GlobalPreferences structs, atomic file I/O, auto-save debounce, and modified `App::resumed()` that loads config before building the grid.
</objective>
