<objective>
Intervention detection that surfaces toast notifications when terminal processes need human attention.

Purpose: When Claude Code asks a permission question or sudo requests a password, the user gets a toast notification immediately — no need to be actively watching that terminal. This is the "AI-native" feature that makes Myco aware of what's happening in its terminal panels. Implements two-layer detection per D-05: pattern matching for known tools PLUS idle-waiting heuristic as fallback.

Output: Background scanner reads terminal grid text, matches against patterns, creates intervention toasts. Idle heuristic catches unknown tools that are waiting for input. Clicking toast focuses source panel. Explicitly dismissing suppresses the pattern for the session.

    **1. Add idle-state tracking to the intervention detector:**

    Per D-05 (locked decision): "Two-layer detection: PTY output pattern matching for known tools PLUS process state idle-waiting heuristic as fallback."

    The idle heuristic catches tools that ask for input but don't use recognizable prompt text. It works by detecting when:
    - The process is in a Sleep/Idle state (from sysinfo ProcessStatus)
    - AND no PTY output has been produced for >5 seconds
    - AND the terminal's last line is not empty (something is displayed that may be a prompt)

    Add to `InterventionDetector`:
    ```rust
    /// Tracks per-panel idle state for the two-layer detection heuristic.
    struct IdleState {
        /// Last time we saw new output (text changed from previous scan)
        last_output_change: Instant,
        /// Previous text hash (or last N chars) to detect output changes
        last_text_hash: u64,
        /// Whether we already fired an idle alert for this idle period
        idle_alert_fired: bool,
    }

    // Add to InterventionDetector:
    idle_states: HashMap<PanelId, IdleState>,
    ```

    **2. Implement idle detection logic in InterventionDetector:**

    Add method:
    ```rust
    /// Check if a panel's process appears to be idle-waiting for input.
    /// Returns Some(InterventionAlert-worthy info) if the heuristic triggers.
    /// 
    /// Conditions (ALL must be true):
    /// 1. Process status is Sleep or Idle (from sysinfo)
    /// 2. No PTY output change for >5 seconds (text hash unchanged)
    /// 3. Terminal's last non-empty line exists (something is displayed)
    /// 4. No pattern match already fired for this scan (avoid double-alerting)
    /// 5. We haven't already fired an idle alert for this idle period
    pub fn check_idle_heuristic(
        &mut self,
        panel_id: PanelId,
        text: &str,
        process_status: Option<ProcessStatus>,
    ) -> Option<(String, String)> {
        // pattern_id = "__idle_heuristic", tool_name = "Process"
        
        let text_hash = Self::hash_text(text);
        let idle_state = self.idle_states.entry(panel_id).or_insert(IdleState {
            last_output_change: Instant::now(),
            last_text_hash: text_hash,
            idle_alert_fired: false,
        });
        
        // Check if text changed
        if text_hash != idle_state.last_text_hash {
            idle_state.last_text_hash = text_hash;
            idle_state.last_output_change = Instant::now();
            idle_state.idle_alert_fired = false; // Reset on new output
            return None;
        }
        
        // Already alerted for this idle period
        if idle_state.idle_alert_fired {
            return None;
        }
        
        // Check idle duration (>5 seconds without output change)
        if idle_state.last_output_change.elapsed() < Duration::from_secs(5) {
            return None;
        }
        
        // Check process status — must be Sleep or Idle
        match process_status {
            Some(ProcessStatus::Sleep) | Some(ProcessStatus::Idle) => {}
            _ => return None, // Process is running/busy, not waiting for input
        }
        
        // Check that the last non-empty line exists (something is displayed)
        let has_visible_prompt = text.lines().rev()
            .find(|line| !line.trim().is_empty())
            .is_some();
        if !has_visible_prompt {
            return None;
        }
        
        // All conditions met — fire idle heuristic alert
        idle_state.idle_alert_fired = true;
        Some(("__idle_heuristic".to_string(), "Process".to_string()))
    }
    
    fn hash_text(text: &str) -> u64 {
        use std::hash::{Hash, Hasher};
        let mut hasher = std::collections::hash_map::DefaultHasher::new();
        text.hash(&mut hasher);
        hasher.finish()
    }
    ```
