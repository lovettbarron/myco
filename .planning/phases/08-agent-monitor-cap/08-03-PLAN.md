   ```rust
   pub fn handle_click(&mut self, x: f32, y: f32, bounds: Rect, is_right_click: bool) -> AgentMonitorAction {
       // Calculate which row was clicked based on y position
       let content_y = y - bounds.y - HEADER_HEIGHT + self.agent_scroll_offset;
       let row_index = (content_y / ROW_HEIGHT) as usize;
       
       if row_index >= self.sessions.len() {
           return AgentMonitorAction::None;
       }
       
       if is_right_click {
           return AgentMonitorAction::ShowContextMenu { row_index, screen_x: x, screen_y: y };
       }
       
       // Check if click is on chevron area (left 24px of row)
       let chevron_x_end = bounds.x + 24.0;
       if x < chevron_x_end {
           if self.expanded_rows.contains(&row_index) {
               self.expanded_rows.remove(&row_index);
               return AgentMonitorAction::CollapseRow(row_index);
           } else {
               self.expanded_rows.insert(row_index);
               return AgentMonitorAction::ExpandRow(row_index);
           }
       }
       
       // Click on the row body = focus terminal
       if let Some(session) = self.sessions.get(row_index) {
           AgentMonitorAction::FocusTerminal(session.source_panel_id)
       } else {
           AgentMonitorAction::None
       }
   }
   ```

   - `ExpandRow`/`CollapseRow` → call `request_redraw()` (state already mutated)
   - `ShowContextMenu` → proceed to task 08-03-03's context menu
   - `None` → no-op

<threat_model>
## Threat Model

| Threat | STRIDE | Severity | Mitigation |
|--------|--------|----------|------------|
| Kill signal sent to wrong PID via context menu | Elevation of Privilege | HIGH | Only send SIGKILL to PIDs verified as children of tracked shell PIDs. Validate in app.rs before dispatching kill. |
| Malformed token patterns causing excessive CPU in parsing | Denial of Service | LOW | Token patterns are simple substring matches (no regex). Linear scan bounded by terminal visible text size (~4KB). |
| Context menu actions on stale session (PID recycled) | Tampering | MEDIUM | Verify PID still exists in sysinfo before sending signals. Check parent is still our shell PID. |
| Clipboard content exfiltration via Copy Stats | Information Disclosure | LOW | Only copies agent stats the user can already see. No sensitive data exposed. |
</threat_model>
